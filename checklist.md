# ✅ Excel Skills Mastery Checklist

Use this checklist to track your progress through all 9 lessons. Check off each skill once you feel confident you can do it on your own — without looking at the instructions!

---

## 📋 How to Use This Checklist

For each skill, ask yourself: *"Could I do this right now without help?"*

- **Not yet** — I haven't tried this / I need to review the lesson
- **Getting there** — I can do it with a little help
- **Got it!** ✅ — I can do it on my own, no problem

---

## 🖥️ Lesson 1 — Getting Started

- [ ] I can open Excel and start a blank workbook
- [ ] I can name the main parts of the Excel screen (cell, row, column, ribbon, formula bar, name box)
- [ ] I can read a cell address — for example, I know that column C, row 4 = **C4**
- [ ] I can click on any cell and see its address in the Name Box
- [ ] I can navigate using keyboard shortcuts (arrow keys, Ctrl+Home, Tab, Enter)
- [ ] I can jump directly to a specific cell by typing its address in the Name Box
- [ ] I can save my file using **Ctrl + S** and give it a name

---

## ✏️ Lesson 2 — Entering and Formatting Data

- [ ] I can type text, numbers, and dates into cells
- [ ] I can edit a cell by double-clicking it or pressing **F2**
- [ ] I can delete a cell's content with the **Delete** key
- [ ] I can undo a mistake with **Ctrl + Z**
- [ ] I can select a range of cells (e.g., A1:B5) using click + Shift+click
- [ ] I can make text **bold**, *italic*, and <u>underlined</u>
- [ ] I can change the background color (fill color) of a cell
- [ ] I can auto-fit a column by double-clicking the border between column letters
- [ ] I can align text (left, center, right) in a cell
- [ ] I can format numbers as Currency, Percentage, or Number

---

## 🔢 Lesson 3 — Basic Formulas

- [ ] I know that every formula starts with an **=** sign
- [ ] I can do basic math in a cell: `=5+3`, `=10-4`, `=6*7`, `=20/4`, `=2^3`
- [ ] I can use cell references in a formula (e.g., `=A1+B1`)
- [ ] I understand that formulas update automatically when the referenced cells change
- [ ] I can use **SUM** to add a range: `=SUM(B2:B6)`
- [ ] I can use **AVERAGE** to find the mean: `=AVERAGE(B2:B6)`
- [ ] I can use **MAX** to find the highest value: `=MAX(B2:B6)`
- [ ] I can use **MIN** to find the lowest value: `=MIN(B2:B6)`
- [ ] I can use **COUNT** to count numeric cells: `=COUNT(B2:B6)`
- [ ] I can use the **AutoSum** button (Σ) to quickly add up numbers
- [ ] I can copy a formula down a column using the **fill handle** (the small + in the corner of a cell)

---

## 📊 Lesson 4 — Charts and Graphs

- [ ] I can explain when to use a bar chart, column chart, line chart, pie chart, or scatter chart
- [ ] I can select a range of data and create a **bar chart** from the Insert tab
- [ ] I can move and resize a chart on the spreadsheet
- [ ] I can add a **title** to a chart
- [ ] I can change the **color scheme** of a chart
- [ ] I can add **data labels** to show the exact value on each bar or slice
- [ ] I can create a **pie chart** and show percentages
- [ ] I can create a **line chart** to show trends over time

---

## 🎮 Lesson 5 — Fun Projects

- [ ] I can build a complete spreadsheet from scratch with headers, data, and formulas
- [ ] I can use **COUNTIF** to count cells that match a specific word or value: `=COUNTIF(E2:E9,"Win")`
- [ ] I can format my spreadsheet so it looks clean and easy to read (bold headers, colors, auto-fit columns)
- [ ] I can create a chart that goes with my data (correct chart type, proper title)
- [ ] I can combine skills from earlier lessons to build a real-world project (game tracker, sports tracker, movie planner, etc.)

---

## 🔢🔤 Lesson 6 — Sorting and Filtering

- [ ] I can sort a column **A to Z** (alphabetical) using the Data tab
- [ ] I can sort a column **Z to A** (reverse alphabetical)
- [ ] I can sort numbers from **largest to smallest** or **smallest to largest**
- [ ] I can do a **multi-level sort** (sort by one column, then by a second column within that)
- [ ] I can turn on **AutoFilter** (Data → Filter) so dropdown arrows appear in column headers
- [ ] I can use a **Number Filter** to show only rows matching a condition (e.g., greater than 7.5)
- [ ] I can use a **Text Filter** to show only rows with a specific word (e.g., only "Animation" movies)
- [ ] I can **clear a filter** to show all rows again
- [ ] I understand that hidden rows still exist — they're just not being shown

---

## 🤔 Lesson 7 — IF Statements

- [ ] I can explain what an IF statement does in plain words
- [ ] I can write a basic IF formula: `=IF(B2>=50,"Pass","Fail")`
- [ ] I know the three parts of IF: **condition**, **value if true**, **value if false**
- [ ] I know to put text values inside **quotation marks** in an IF formula
- [ ] I can use all comparison operators: `>`, `<`, `>=`, `<=`, `=`, `<>`
- [ ] I can write a **nested IF** to create multiple categories (e.g., A, B, C, D grades)
- [ ] I can use the **IFS** function as a cleaner alternative to nested IFs (Excel 2019+)
- [ ] I can use **COUNTIF** to count cells matching a condition: `=COUNTIF(B2:B11,">=90")`

---

## 🔍 Lesson 8 — VLOOKUP

- [ ] I can explain in plain words what VLOOKUP does (it searches the first column of a table and returns a value from another column in the same row)
- [ ] I know the four parts of VLOOKUP: **lookup value**, **table array**, **column number**, **TRUE/FALSE**
- [ ] I always use **FALSE** for exact matching
- [ ] I can write a VLOOKUP to find a value: `=VLOOKUP(E2, A1:C6, 2, FALSE)`
- [ ] I can change the column number to look up different pieces of information from the same table
- [ ] I can use **Sheet1!** in my formula to look up data from a different sheet: `=VLOOKUP(B1, Sheet1!A:G, 2, FALSE)`
- [ ] I can wrap a VLOOKUP in **IFERROR** so a friendly message appears instead of `#N/A`: `=IFERROR(VLOOKUP(...),"Not found")`

---

## 🎨 Lesson 9 — Conditional Formatting

- [ ] I can explain what Conditional Formatting does (it automatically changes a cell's appearance based on its value)
- [ ] I can highlight cells **greater than** a value in a chosen color
- [ ] I can highlight cells **less than** a value in a chosen color
- [ ] I can highlight cells **between** two values in a chosen color
- [ ] I can apply multiple Conditional Formatting rules to the same range
- [ ] I can apply a **Color Scale** so cells automatically shade from red (low) to green (high)
- [ ] I can add **Data Bars** to cells so they show a mini bar chart inside the cell
- [ ] I can apply **Icon Sets** (traffic lights, stars, arrows) to cells based on their values
- [ ] I can write a **custom formula rule** (e.g., `=$H2>=4`) to highlight entire rows
- [ ] I understand why `$H2` (dollar sign before the column) is needed in row-highlighting formulas
- [ ] I can use **Manage Rules** to view, edit, reorder, or delete Conditional Formatting rules
- [ ] I can **clear all rules** from a selected range
