# Lesson 1: Getting Started with Excel 🖥️

**⏱️ Time:** About 20 minutes  
**📚 What you'll learn:** How to open Excel, understand the screen, and navigate around

---

## Step 1 — Opening Excel

1. Click the **Start Menu** (the Windows logo button) on your computer.
2. Type **"Excel"** in the search bar.
3. Click on **Microsoft Excel** to open it.
4. When Excel opens, click **"Blank Workbook"** to start a new, empty spreadsheet.

> 💡 **On a Mac?** Click the **Finder** icon → **Applications** → **Microsoft Excel**

---

## Step 2 — Understanding the Excel Screen

When Excel opens, you'll see a lot going on! Let's break it down:

```
┌─────────────────────────────────────────────────────────────────┐
│  📌 TITLE BAR    →  Shows the file name (e.g., "Book1 - Excel") │
├─────────────────────────────────────────────────────────────────┤
│  🎛️ RIBBON       →  Buttons and menus at the top (Home, Insert…)│
├───────────┬─────────────────────────────────────────────────────┤
│  🔢 NAME  │  📝 FORMULA BAR  →  Shows what's in a selected cell │
│  BOX      │                                                      │
├───────────┴─────────────────────────────────────────────────────┤
│  📋 THE SPREADSHEET  →  A giant grid of rows and columns        │
│                                                                  │
│     A       B       C       D                                    │
│  ┌──────┬───────┬───────┬───────┐                               │
│1 │      │       │       │       │   ← ROW 1                     │
│  ├──────┼───────┼───────┼───────┤                               │
│2 │      │       │       │       │   ← ROW 2                     │
│  ├──────┼───────┼───────┼───────┤                               │
│3 │      │       │       │       │                               │
│  └──────┴───────┴───────┴───────┘                               │
│   ↑       ↑                                                      │
│  COL A   COL B                                                   │
│                                                                  │
├─────────────────────────────────────────────────────────────────┤
│  📄 SHEET TABS   →  "Sheet1", "Sheet2"… at the bottom           │
└─────────────────────────────────────────────────────────────────┘
```

### Key Parts to Know:

| Part | What It Does |
|------|-------------|
| **Cell** | A single box in the grid (like A1, B3) |
| **Row** | A horizontal line of cells (1, 2, 3…) |
| **Column** | A vertical line of cells (A, B, C…) |
| **Ribbon** | The menu bar with all the tools |
| **Formula Bar** | Shows the content of the selected cell |
| **Name Box** | Shows the address of the selected cell |

## Step 3 — Understanding Cell Addresses

Every cell has an **address** made of a **column letter** + **row number**.

**Examples:**
- The cell in column **A**, row **1** → **A1**
- The cell in column **C**, row **5** → **C5**
- The cell in column **B**, row **3** → **B3**

### Try It!
1. Click on a cell in the spreadsheet.
2. Look at the **Name Box** (top-left corner). It shows the cell address!
3. Click on the cell in column **D**, row **7**. What does the Name Box say? (**D7**)

## Step 4 — Navigating the Spreadsheet

You can move around in several ways:

| Action | How To Do It |
|--------|-------------|
| Move one cell | Press **Arrow Keys** (↑ ↓ ← →) |
| Jump to a cell | Type the address in the **Name Box** and press **Enter** |
| Move to the right | Press **Tab** key |
| Move down | Press **Enter** key |
| Go to cell A1 | Press **Ctrl + Home** |
| Go to the last used cell | Press **Ctrl + End** |

### Try It!
1. Press **Ctrl + Home** to go to cell **A1**.
2. Press the **right arrow** 5 times. Where are you? (**F1**)
3. Press the **down arrow** 3 times. Where are you? (**F4**)
4. Click the **Name Box**, type **B2**, and press **Enter**. Did you jump to B2?

---

## Step 5 — The Ribbon (Your Toolbox!)

The **Ribbon** at the top has tabs full of tools. The main ones are:

| Tab | What's In It |
|-----|-------------|
| **Home** | Font style, size, colors, alignment |
| **Insert** | Charts, images, tables |
| **Page Layout** | Margins, orientation, print settings |
| **Formulas** | All the math functions |
| **Data** | Sorting and filtering |
| **View** | How the spreadsheet looks on screen |

### Try It!
1. Click the **Insert** tab. Look at what's there!
2. Click the **Home** tab to go back.

---

## Step 6 — Saving Your Work

**Don't lose your work!** Save often using:

- **Ctrl + S** (Windows) or **Cmd + S** (Mac) — Save quickly
- Click **File** → **Save As** — Save with a new name

### Try It!
1. Press **Ctrl + S**.
2. Give your file the name **"Excel Practice"**.
3. Choose where to save it (like your Desktop or Documents).
4. Click **Save**.
