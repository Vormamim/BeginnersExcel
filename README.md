# BeginnersExcel

A beginner-friendly introduction to Microsoft Excel for new users.

## About

This repository contains learning materials, exercises, and sample data to help beginners get started with Excel.

## Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/Vormamim/BeginnersExcel.git
   ```
2. Open the lesson files in your browser or text editor.
3. Follow along with the exercises using Microsoft Excel or any compatible spreadsheet application (e.g., Google Sheets, LibreOffice Calc).
